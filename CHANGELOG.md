# Changelog

## v0.1.2 - 2026-07-02

Agrega la capa comun de coordinacion para conectar la realidad ya observada: grupos de WhatsApp, mesas de datos, repositorios tecnicos, hojas, apps, APIs y fuentes operativas.

Incluye:

- Documento `docs/common-coordination-layer-v0.1.md`.
- Schema `schemas/common_coordination_item_schema.csv`.
- Schema `schemas/common_operational_signal_schema.csv`.

La capa comun no reemplaza proyectos existentes. Define un idioma minimo para conectar funciones, zonas, tiempos, estados, fuentes, permisos y riesgos sin centralizar personas.

## v0.1.1 - 2026-07-02

Agrega materiales practicos para que personas, organizaciones y equipos tecnicos puedan empezar a compartir datos de forma segura.

Incluye:

- Guia rapida para compartir datos seguros.
- Checklist de calidad de datos operativos.
- Plantilla tipo Google Sheet/CSV.
- API contract minimo para interoperabilidad segura.
