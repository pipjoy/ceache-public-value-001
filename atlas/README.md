# Atlas 001

`platform_function_matrix.csv` clasifica 168 fuentes, apps, hojas, APIs y sistemas por funcion observable.

## Que significa

La matriz ayuda a observar:

- funciones repetidas;
- posibles redundancias;
- riesgos PII;
- senales de interoperabilidad;
- tipos de datos que podrian mapearse de forma segura.

## Que no significa

La presencia de una plataforma en la matriz no implica:

- afiliacion con Ceache Lab;
- recomendacion;
- auditoria tecnica completa;
- impacto social probado;
- vigencia garantizada;
- disponibilidad real de datos.

Es un inventario de funciones observables, no un ranking.
