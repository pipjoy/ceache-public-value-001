# Schemas CSV

Estos schemas son contratos minimos para compartir datos operativos sin exponer personas.

Todos los archivos usan estas columnas:

```text
field,type,required,description,example,pii_risk,public_allowed
```

## Schemas incluidos

- `acopios_schema.csv`
- `donaciones_schema.csv`
- `necesidades_schema.csv`
- `refugios_schema.csv`
- `voluntarios_schema.csv`
- `entregas_schema.csv`
- `source_catalog_schema.csv`

## Regla de uso

Si tu campo contiene nombres, cedulas, telefonos privados, salud individual, menores, personas desaparecidas, ubicaciones exactas sensibles o chats privados, no lo publiques.

Usa datos agregados y operativos.
