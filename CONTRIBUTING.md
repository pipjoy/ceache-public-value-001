# Contribuir datos o mejoras

Gracias por querer contribuir.

Este repositorio no recibe datos personales. Si quieres compartir una lista, hoja, app o API, primero revisa `DATA_SAFETY.md`.

## Puedes contribuir

- mapeos de columnas a los schemas;
- mejoras de definiciones;
- ejemplos anonimizados;
- documentacion tecnica;
- correcciones metodologicas;
- nuevas categorias operativas;
- propuestas de interoperabilidad segura.

## No contribuyas

- datos personales;
- bases nominales;
- chats;
- imagenes de personas;
- ubicaciones exactas sensibles;
- datos de salud individual;
- informacion de menores;
- personas desaparecidas.

## Como proponer cambios

1. Abre un issue explicando que quieres mejorar.
2. Indica si tu propuesta afecta documentos, schemas o atlas.
3. Si abres un pull request, explica el cambio y el riesgo de datos.
4. Mantén el principio: interoperar funciones, no personas.
